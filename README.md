## Описание
Одностраничный сайт-портфолио с представленными учебными и коммерческими проектами.  
**Дизайн и функционал разработаны самостоятельно.**  
Основная цель проекта — продемонстрировать навыки вёрстки, адаптивного дизайна, работы с React и современной frontend-сборкой.

## Установка и запуск
```bash
# Клонировать репозиторий
git clone https://github.com/Vinlasy/my-portfolio.git

# Перейти в директорию проекта
cd my-portfolio

# Установить зависимости
npm install

# Запустить проект в режиме разработки
npm run dev
